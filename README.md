# HR Attrition Risk Analysis Dashboard

## Problem Statement
Companies lose crores every year when employees resign unexpectedly. 
This project predicts which employees are most likely to quit.

## Dataset
IBM HR Analytics Dataset — 1,470 employees, 35 columns

## Tools Used
- Python (pandas, matplotlib)
- SQL (SQLite)
- Power BI

## Key Findings
- Sales department has highest attrition — 20.6%
- Employees who left earned 30% less salary
- Overtime workers are 3x more likely to quit
- Employees under 30 leave the most

## Dashboard Features
- KPI Cards — Total left, Avg salary, Attrition rate
- Interactive slicer — filter by department
- Risk score model — predicts at-risk employees
- Conditional formatting — green to red risk scale

## Files
- day1.py to day9.py — Python analysis code
- hr_final.csv — cleaned dataset with risk scores
- HR_Attrition_Report.pdf — 6 page analysis report
- HR_Dashboard.pbix — Power BI dashboard
